# Spotify Song Popularity Classification

This project investigates whether Spotify audio features can be used to classify whether a song is popular using machine learning techniques.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Danceability
- Energy
- Loudness
- Speechiness
- Acousticness
- Instrumentalness
- Liveness
- Valence
- Tempo

## Results
Random Forest achieved the strongest performance with approximately 85% accuracy and an ROC-AUC score of 0.79.
